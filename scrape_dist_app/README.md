## 1. 시작 전 준비

### 구글 서비스 계정 키 파일

프로그램이 구글 시트에 접근하려면 서비스 계정 키 파일이 필요합니다.

**파일 위치**

키 파일은 반드시 아래 경로에 놓아야 합니다.

```
최상위폴더/
└── credentials/
    └── google_key.json   ← 여기에 넣으세요
```

> 키 파일 이름이 다르거나 위치가 다르면 프로그램이 실행되지 않습니다.  
> 파일명은 반드시 `google_key.json` 이어야 합니다.

## 2. 설정 방법

프로그램 우측 하단의 **⚙ 버튼**을 눌러 환경 설정 창을 엽니다.

### 구글 시트 URL 입력

**시트 URL 찾는 방법:**

1. 브라우저에서 대상 구글 스프레드시트를 엽니다.
2. 주소창의 URL 전체를 복사합니다.

   ```
   https://docs.google.com/spreadsheets/d/1zYxXyyDXI.../edit#gid=0
   ```

3. 환경 설정 창의 **시트 URL** 입력란에 붙여넣습니다.

> URL 중간의 긴 문자열(`1zYxXyyDXI...`)이 시트 고유 ID입니다.  
> `#gid=` 이후 부분이 포함되어 있어도 괜찮습니다.

### 테마 선택

- **다크 🌙** : 어두운 배경 테마
- **라이트 ☀️** : 밝은 배경 테마

설정 후 **저장** 버튼을 누르면 즉시 적용됩니다.

## 3. 사용 방법

### 1단계 — 엑셀 파일 선택

**파일 선택** 버튼을 눌러 작업할 원본 엑셀 파일을 선택합니다.

- 지원 형식: `.xlsx`, `.xlsm`
- `.xls` 파일은 지원하지 않습니다.

### 2단계 — 종류 선택

오른쪽 라디오 버튼에서 작업 대상을 선택합니다.

| 항목 | 구글 시트 탭 |
|------|-------------|
| 서울 | 첫 번째 시트 |
| 부산 | 두 번째 시트 |
| 디파 | 세 번째 시트 |

### 3단계 — 작업 범위 입력

처리할 박스 번호 범위를 입력합니다.

- **시작 박스** : 예) `B0001`
- **종료 박스** : 예) `B0050`

### 4단계 — 실행

**▶ 실행** 버튼을 누르면 작업이 시작됩니다.

진행 상황 토글을 클릭하면 로그 창에서 단계별 진행 상태를 확인할 수 있습니다.

---

## 4. 작업 결과

작업이 완료되면 원본 엑셀 파일과 동일한 폴더에 결과 파일이 생성됩니다.

```
원본 파일명_자동분류.xlsx
```

예시: `인쇄자료.xlsx` → `인쇄자료_자동분류.xlsx`

작업 완료 후 결과 파일이 자동으로 열립니다.

---

## 5. 오류 대처

| 오류 메시지 | 원인 | 해결 방법 |
|------------|------|-----------|
| 구글 서비스 계정 키 파일을 찾을 수 없습니다 | `credentials` 폴더에 `google_key.json` 파일 없음 | `google_key.json` 파일을 `credentials` 폴더에 넣었는지 확인 |
| 유효하지 않은 스프레드시트 URL | 환경 설정의 URL 입력이 잘못됨 | 올바른 구글 시트 URL을 복사해 붙여넣기 |
| 스프레드시트를 찾을 수 없습니다 | 시트 접근 권한 없음 | 서비스 계정 이메일에 시트 편집 또는 뷰 권한 부여 |
