## 1. 시작 전 준비

### 허브에서 구글 로그인

이 프로그램은 별도의 서비스 계정 키 파일을 사용하지 않습니다. 대신 **허브(`hub.py`)에 로그인된 구글 계정**을 공유하여 시트에 접근합니다.

1. 허브(`hub.py`)를 실행합니다.
2. 허브 화면의 **구글 계정** 영역에서 **로그인**을 눌러 구글 계정으로 로그인합니다.
3. 로그인하면 OAuth 토큰이 **Windows 자격 증명 관리자**에 암호화(DPAPI) 저장되어, 이 프로그램이 자동으로 같은 세션을 사용합니다.

> 허브에 로그인하지 않은 상태로 실행하면 *"구글 계정 로그인이 필요합니다. hub에서 먼저 로그인해 주세요."* 오류가 발생합니다.  
> 로그인은 허브·백업툴·이 프로그램이 모두 공유하므로, 한 곳에서만 로그인하면 됩니다.

## 2. 설정 방법

프로그램 우측 하단의 **⚙ 버튼**을 눌러 환경 설정 창을 엽니다.

### 구글 시트 URL 입력

**시트 URL 찾는 방법:**

1. 브라우저에서 대상 구글 스프레드시트를 엽니다.
2. 주소창의 URL 전체를 복사합니다.

   ```
   https://docs.google.com/spreadsheets/d/1zYxXyyDXI.../edit#gid=0
   ```

3. 환경 설정 창의 **시트 URL** 입력란에 붙여넣습니다.

> URL 중간의 긴 문자열(`1zYxXyyDXI...`)이 시트 고유 ID입니다.  
> `#gid=` 이후 부분이 포함되어 있어도 괜찮습니다.

### 테마 선택

- **다크 🌙** : 어두운 배경 테마
- **라이트 ☀️** : 밝은 배경 테마

설정 후 **저장** 버튼을 누르면 즉시 적용됩니다.

## 3. 사용 방법

### 1단계 — 엑셀 파일 선택

**파일 선택** 버튼을 눌러 작업할 원본 엑셀 파일을 선택합니다.

- 지원 형식: `.xlsx`, `.xlsm`
- `.xls` 파일은 지원하지 않습니다.

### 2단계 — 종류 선택

오른쪽 라디오 버튼에서 작업 대상을 선택합니다.

| 항목 | 구글 시트 탭 |
|------|-------------|
| 서울 | 첫 번째 시트 |
| 부산 | 두 번째 시트 |
| 디파 | 세 번째 시트 |

### 3단계 — 작업 범위 입력

처리할 박스 번호 범위를 입력합니다.

- **시작 박스** : 예) `B0001`
- **종료 박스** : 예) `B0050`

### 4단계 — 실행

**▶ 실행** 버튼을 누르면 작업이 시작됩니다.

진행 상황 토글을 클릭하면 로그 창에서 단계별 진행 상태를 확인할 수 있습니다.

---

## 4. 작업 결과

작업이 완료되면 원본 엑셀 파일과 동일한 폴더에 결과 파일이 생성됩니다.

```
원본 파일명_자동분류.xlsx
```

예시: `인쇄자료.xlsx` → `인쇄자료_자동분류.xlsx`

작업 완료 후 결과 파일이 자동으로 열립니다.

## 5. 오류 대처

| 오류 메시지 | 원인 | 해결 방법 |
|------------|------|-----------|
| 구글 계정 로그인이 필요합니다. hub에서 먼저 로그인해 주세요 | 허브에 로그인된 구글 계정이 없음 | `hub.py`를 실행하고 구글 계정으로 로그인 |
| 허브 토큰 갱신 실패. hub.py 에서 다시 로그인해 주세요 | 저장된 토큰이 만료·폐기됨 | 허브에서 로그아웃 후 다시 로그인 |
| 유효하지 않은 스프레드시트 URL | 환경 설정의 URL 입력이 잘못됨 | 올바른 구글 시트 URL을 복사해 붙여넣기 |
| 스프레드시트를 찾을 수 없습니다 | 로그인한 계정에 시트 접근 권한 없음 | 로그인한 구글 계정에 시트 편집 또는 뷰 권한 부여 |
